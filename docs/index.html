Installation

Get up and running in seconds

Recommended

From Source

Clone and build 
$ git clone https://github.com/only-using-ai/rustxl
$ cd rustxl
$ cargo build --release
Add to PATH (optional) 
$ cp target/release/xl /usr/local/bin/

With Cargo

Install directly 
$ cargo install --path .

Requires Rust 1.85+ (2024 edition)

Usage

Simple, intuitive, powerful

Open a file

$ xl spreadsheet.csv
$ xl data.xlsx
$ xl report.tsv

Create new spreadsheet

$ xl

Pipe data in

$ ls -la | xl
$ cat data.csv | xl
$ curl api.example.com | xl

Process and save

# Open file, edit, press 's' to save
$ xl input.csv
# Save as new file in save dialog

Formula Reference

Excel-compatible functions for all your calculations

Math Functions

=SUM(A1:A10) Sum of values in range
=AVG(A1:A10) Average of values
=MIN(A1:A10) Minimum value
=MAX(A1:A10) Maximum value
=COUNT(A1:A10) Count numeric cells
=COUNTA(A1:A10) Count non-empty cells
=ROUND(A1, 2) Round to decimal places
=CORREL(A1:A10, B1:B10) Correlation coefficient

Conditional Functions

=IF(A1>10, "Yes", "No") Conditional logic
=COUNTIF(A1:A10, ">5") Count matching criteria
=SUMIF(A1:A10, ">5") Sum matching criteria
=AVERAGEIF(A1:A10, ">5") Average matching criteria

Text Functions

=CONCATENATE(A1, B1) Join text values
=CONCAT(A1, " ", B1) Join text (alias)
=LEFT(A1, 5) First N characters
=RIGHT(A1, 5) Last N characters
=MID(A1, 2, 5) Substring from position
=LEN(A1) Length of text

Lookup & Special

=VLOOKUP(A1, B1:D10, 2) Vertical lookup
=SHELL("date") Execute shell command

Formulas start with = and support arithmetic operators (+, -, *, /) with proper precedence. Cell references use A1 notation.
